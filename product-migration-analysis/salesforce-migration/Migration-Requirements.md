# Salesforce Migration Requirements
- This will be a one-time migration of Orienteer to SalesForce SaaS
- Downtime for the one-time migration is expected.
- It is expected that OrientDB class/table definitions and needed relationships will be gathered and provided.
- Graph relationship data does not need to be migrated.
- The entire OrientDB database is less than 500 MB
- the Salesforce installation target is clean and contains no existing data
- The migration should be automated as we expect several test migrations so users can verify functionality in a non-production environment.
